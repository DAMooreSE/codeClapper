HOW TO RUN:

ADD RULE TO FIREWALL FOR NETWORK-WIDE ACCESS:
    1. Allow access from inbound unsecured devices on port 8080

RUN START SCRIPT
    1. Open cmd and navigate to the projects directory
    2. Type and execute the line "npm install" to install the necessary packages through npm
    3. Type and execute the line "npm run server"
    4. The program is now hosted on port 8080 of the machine and can be accessed through localhost:8080 in a web browser
